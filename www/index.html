LIGHTWEIGHT & SIMPLE

Fend's core contains ~200LoC (including basic features). The library is made out of 3 simple classes.

EXTENSIBLE

Modeled after Roda's plugin system. All features are provided through plugins, which you can pick based on your needs. If you are not happy with provided solution, create new plugin to add new, or modify existing functionality.

NO CONSTRAINTS

No decisions are made for you in advance. Altough common validation scenarios are covered (through plugins), you are stil in control when it comes to validation logic.
